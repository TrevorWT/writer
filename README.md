# Writer

Plan and write your book in one place. Zoom from "what's my story about" down
to the sentence you're writing — outline, notes, and prose are the same
document, saved as plain Markdown files you'll never be locked out of.

**[⬇ Download the latest version](https://github.com/TrevorWT/writer/releases/latest)**
(Windows — grab the `Writer_x.x.x_x64-setup.exe` file under Assets)

> **SmartScreen warning?** The alpha isn't code-signed yet, so Windows will
> warn on first run. Click **More info → Run anyway**. The app is ~2 MB, works
> completely offline, and only touches the folder you pick for your stories.

## What it does

- **The outline is the manuscript** — ctrl+scroll to zoom between story, parts,
  chapters, and scenes. Brainstorm at any level; planning notes never leak
  into your exported manuscript.
- **Structure names itself** — start with scenes, nest as you grow; layers
  become Chapters and Parts automatically.
- **Story bible as a side effect** — highlight a name, tag it as a character
  (or location, or item); every mention becomes a link with backlinks.
- **Preview & export** — read the compiled story in-app, or export a
  Standard-Manuscript-Format .docx, HTML, or Markdown.
- **Your files, forever** — everything is plain Markdown in a folder you
  choose. Automatic backups and snapshots included. No account, no cloud,
  no lock-in.
